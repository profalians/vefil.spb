== Description ==
This plugin allows you to set datepicker,calendar on your posts and pages just in single click. Ability to change settings of calendar according to your requirement.

Importance

Using calendar,datepicker in posts and pages is little bit tricky to newly wordpress users because integration of jquery UI Calendar makes some jQuery confliction and time consuming task.So This Plugin help users to make their DATE Field more funky and stylish just in one single click,and also have power to user set their DATE Format setting according to their requirements.

Usage:

Once activated, make sure to configure the plugin in the settings menu with the one of the available options.

1. Calendar Picker

and finally, you're able to change settings of calendar need to show on posts and pages.
If You dont wants to use setting of calendar datepicker, you just only use "calendarpicker" as your class attribute the Default settings automatically applied on your field in one single click.

Notes:

This plugin for WordPress, the publicly available  If you experience any problems with the plugin, feel free to contact me.

Please please please don't forgot to rate this plugin if you found it works and useful for your blog.


== Installation ==
This section describes how to install the plugin and get it working.
	
	1.	Upload the calendar-picker directory to the /wp-content/plugins/ directory

	2.	Activate the plugin through the 'Plugins' menu in WordPress

	3.	Change the calendar picker settings according to your requirement and use "calendarpicker" as your class attribute.

== Frequently Asked Questions ==

= Why should i install this plugin? Will does it help my blog in calendar setting ? = 

Using this plugin you can gives emphasis to the cretins DATE fields Like DOB etc.User Friendly Date Field where user can select date easily to avoid typing wrong date.

= Can i use datepicker multiple times on my blog post or page ? = 

Yes, You can use unlimited numbers of calendar , datepicker on your blog posts or page, But please ensure that you don't overdo it,just use this word "calendarpicker" as your class attribute.Wherever you use this as class attribute Default functionality of Calendar Datepicker applied.
